Необходимо создать Калькулятор, используя модульную систему.

1. создать модуль "calculator". внутри него создать класс 
Calculator и определить методы необходимые для калькулятора 

2. создать главный файл main.py куда будет импортирован 
модуль Калькулятор и будут производиться действия.

в данном файле main.py должны быть выполнены:

- импортировать модуль random (почитайте про него)

- создать список rand_list из 20 случайных чисел используя
 модуль random (Числа в данном списке должны 
    быть целочисленные от 0 до 100 - не float)

- Инициализация класса Калькулятор (т.е. создать объект)

- показать выполнение 4 действий через объект класса -
сложение, вычитание, умножение, деление.
    числа для этих действий будут браться в с
    лучайном порядке из списка rand_list


Особенности:
    -   метод сложения должен принимать неограниченное число значений и уметь складывать их все.
      например:
        calc.add(1,59,80,45) или calc.add(1,5) - оба этих варианта должны работать и делать вычисления.
